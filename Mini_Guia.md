----------GUIA PARA YO ENTENDER COMO AGREGAR UN ARCHIVO A GITHUB-------------------

primero hay que clonar el repositorio con el que vamos a trabajar 

## git clone https://github.com/Eviix90s/CiberGenIA.git

Luego agregamos el archivo de manera local 

## git add Urban_Cats.pdf

Después se agrega un Commit con el mensaje de la modificación

## git commit -m "Agregar el archivo Urban_Cats.pdf"

Por ultimo se hace un push para agregarlo al repositorio

## git push origin main
